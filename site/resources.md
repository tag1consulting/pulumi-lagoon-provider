# Resources

The Pulumi Lagoon Provider includes 18 resources organized into six categories. Each resource corresponds to an object managed through the Lagoon GraphQL API.

| Resource | Description | Documentation |
|----------|-------------|---------------|
| Project | Lagoon projects (applications/sites) | [Core Resources](core) |
| Environment | Environments (branch/PR deployments) | [Core Resources](core) |
| Variable | Project and environment variables | [Core Resources](core) |
| Group | Groups for organizing projects and users | [Core Resources](core) |
| DeployTarget | Kubernetes cluster deploy targets | [Deploy Targets](deploy-targets) |
| DeployTargetConfig | Branch-pattern routing to deploy targets | [Deploy Targets](deploy-targets) |
| NotificationSlack | Slack deployment notifications | [Notifications](notifications) |
| NotificationRocketChat | RocketChat deployment notifications | [Notifications](notifications) |
| NotificationEmail | Email deployment notifications | [Notifications](notifications) |
| NotificationMicrosoftTeams | Microsoft Teams deployment notifications | [Notifications](notifications) |
| ProjectNotification | Link notifications to projects | [Notifications](notifications) |
| Route | API-managed routes (requires Lagoon v2.29.0+) | [Routes](routes) |
| ProjectAutogeneratedRouteConfig | Project-level autogenerated route configuration | [Routes](routes) |
| EnvironmentAutogeneratedRouteConfig | Environment-level autogenerated route configuration | [Routes](routes) |
| Task | Advanced task definitions (command and image types) | [Tasks](tasks) |
| User | Lagoon users | [Users](users) |
| UserGroupAssignment | User-to-group role assignments | [Users](users) |
| UserPlatformRole | Platform-level role assignments | [Users](users) |
