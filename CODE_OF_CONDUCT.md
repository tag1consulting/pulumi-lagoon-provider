# Code of Conduct

All contributors and participants in this project are expected to follow these guidelines.

## Principles

1. **Consider the needs of others** — Be patient, kind, and mindful of impact. Value all contributions equally.
2. **Treat each other with respect, even when you disagree** — No personal attacks. Disagreement is not license for poor behavior.
3. **Be collaborative** — Work in the open. Accept responsibility for mistakes. Give and receive feedback gracefully.
4. **Do not tolerate abusive behavior** — Harassment, intimidation, threats, discriminatory language, and unwelcome sexual attention are not acceptable in any form.
5. **Ask for help when you need it** — Check existing documentation before asking. Help keep docs accurate and up to date.
6. **Step down considerately** — If you need to disengage, do so in a way that lets others pick up where you left off.

## Scope

These expectations apply to all project spaces: GitHub issues, pull requests, code review comments, and any other project-related communication.

## Reporting

Report violations by opening a [GitHub issue](https://github.com/tag1consulting/pulumi-lagoon-provider/issues) or emailing [greg@tag1consulting.com](mailto:greg@tag1consulting.com). Reports will be handled with discretion.
